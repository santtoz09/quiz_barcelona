# Quiz Barcelona

Site estático de quiz com tema do Barcelona, baseado no mockup enviado. Abra `index.html` no navegador para jogar.

## Funcionalidades

- Tela inicial com título “Quiz Barcelona” e botão para iniciar.
- Cinco perguntas com respostas em branco antes da escolha.
- Feedback visual: resposta escolhida fica verde quando correta e vermelha quando incorreta.
- Transição animada entre perguntas, com saída pela esquerda e entrada pela direita.
- Tela final com pontuação.
